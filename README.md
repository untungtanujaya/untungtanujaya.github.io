# personal-website-v3

Astro portfolio site for Untung Tanujaya. This version keeps the same route and content architecture as `personal-website-v2`, with a refreshed dark UI, improved mobile layout, and Tailwind compiled utilities for the React clinical apps.

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Structure

- `src/pages/` contains the Astro routes.
- `src/data/` contains resume and project data.
- `src/content/articles/` contains technical articles.
- `src/components/` contains the React clinical app surfaces.
- `src/styles/global.css` contains design tokens, portfolio components, and form/reader styles.
